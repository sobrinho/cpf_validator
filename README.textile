h1. ValidatesAsCpf

This gem is based on *OSS* works in "https://projetos.ossystems.com.br/projects/plugonrails/repository/browse/plugins/validates_as_cpf":https://projetos.ossystems.com.br/projects/plugonrails/repository/browse/plugins/validates_as_cpf

h2. Installation

You can install as gem:

<pre>gem "validates_as_cpf"</pre>

h2. Usage

<pre>class Person < ActiveRecord::Base
  validates :document, :cpf => true
end</pre>
